## ✨ 核心特性

- 🧠 **LangGraph驱动**: 基于状态图的Agent编排引擎
- 🔧 **工具集成**: 支持LangChain和自定义工具
- 🌐 **REST API**: 标准HTTP接口，支持多语言客户端
- 📦 **模块化设计**: 分层架构，易于扩展和维护
- 🐳 **容器化**: Docker原生支持
- 📊 **可观测性**: 结构化日志和监控


## 🎯 支持的LLM

- OpenAI GPT-4 / GPT-3.5
- Anthropic Claude
- SiliconFlow (兼容OpenAI API)
