# 数据分析技能

## 概述
本技能指导如何使用工具进行数据分析和统计计算。

## 使用步骤
1. 准备数据：确保数据格式正确
2. 选择分析工具：根据需求选择合适的工具
3. 执行分析：使用 calculator 工具进行计算
4. 生成报告：整理分析结果

## 工具使用指南
- **calculator**: 用于数值计算和统计分析
  - 支持基本运算：加减乘除
  - 支持函数：sqrt, sin, cos, log 等
  - 支持常量：pi, e

## 最佳实践
- 先进行数据清洗
- 选择合适的统计方法
- 验证计算结果
